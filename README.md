# QuestFlow — Хроники Судьбы

Планировщик задач в стиле тёмного фэнтези. Превращает повседневные дела в эпические RPG-квесты.

## ⚔ Что это

QuestFlow — это PWA (Progressive Web App), который геймифицирует управление задачами в эстетике Dragon Age и Dark Souls:

- **Квесты вместо задач** — каждое дело получает фэнтезийное название и описание
- **Прокачка героя** — XP, золото, уровни, классы (Воин Дел, Хранитель Знаний и др.)
- **Подземелье** — пошаговая боёвка с врагами и боссами
- **Обряды (рутины)** — повторяющиеся задачи со счётчиками серий
- **Подземелье судьбы** — пошаговая RPG-боёвка
- **Календарь, Архив, Журнал побед** — полный набор инструментов планировщика
- **Mood Tracker** — настроение влияет на атмосферу
- **Боевая Концентрация** — Pomodoro 25 минут со звуковым сопровождением

## 🚀 Технологии

- Чистый HTML + CSS + JavaScript (без фреймворков)
- localStorage для оффлайн-данных
- Service Worker для PWA
- Canvas API для процедурной генерации артов
- Опционально: Claude API для AI-генерации текста квестов

## 📱 Установка

Открой ссылку на телефоне или ПК → в меню браузера выбери "Добавить на главный экран" / "Установить приложение". Работает офлайн.

## 🛠 Локальная разработка

```bash
# Запустить локальный сервер
npx serve -p 8765

# Открыть http://localhost:8765
```
